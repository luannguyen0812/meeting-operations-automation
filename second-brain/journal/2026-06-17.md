# Session Summary — 2026-06-17

## What We Built

Full meeting automation system for Luan's 7 Intrastack projects, plus Telegram channel reminders.

### Meeting Automation (Night-Before Workflow)

- Generates .docx agenda + minutes templates per project
- Uploads to Google Drive, archives old files
- Pulls team roster from Google Sheet for attendee lists
- Cron job `meeting-templates-nightly` runs 10PM ET daily (calendar-aware, skips days with no meetings)
- Post-meeting workflow is manual: Luan DMs "Sparky, [project] done" → transcript fetch → summarize → email minutes

### Telegram Channel Reminders

- Script polls Google Calendar every 5 min, caches results for 15 min
- Sends "⚠️ [Meeting] has been cancelled" if a meeting disappears from calendar
- Sends "[Meeting] is starting in 15'" 15 minutes before start
- Messages appear as "Intrastack Communication Channel" (bot added as anonymous admin)
- Bot display name changed via BotFather to something neutral (was @LuanAssistanceGuru_bot)
- Zero Anthropic token cost — pure Python hitting Google Calendar + Telegram APIs directly

## Key Decisions

- Direct SMTP via Mailcow (dropped Zapier)
- Python 3.9 (/usr/bin/python3) — 3.15 alpha breaks cryptography
- Calendar-aware scheduling instead of hardcoded weekdays
- Manual trigger for post-meeting, automated for pre-meeting
