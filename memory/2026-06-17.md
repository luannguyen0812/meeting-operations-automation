- `scripts/meeting-automation/config.py` — project data, credentials, agenda template

- `scripts/meeting-automation/send_email.py` — sends via direct SMTP (Mailcow)
- `scripts/meeting-automation/calendar_check.py` — checks Google Calendar for tomorrow's meetings
- `scripts/meeting-automation/workflow_nightbefore.py` — orchestrates template gen + upload + cleanup
- `scripts/meeting-automation/workflow_postmeeting.py` — orchestrates transcript → name fix → summarize → email
